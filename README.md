## Bash keyboard shortcuts

`ctrl + a` - go to beginning of line
`ctrl + e` - go to end of line
`ctrl + l` - similar to running `clear` command
`ctrl + k` - clear entire line upto the cursor
`ctrl + w` - delete the last word
